# 📊 Student Management System (SQL Project)

## 📌 Overview
This project demonstrates core SQL concepts including:
- Database creation
- CRUD operations
- Data generation (1000 records)
- Joins and relational queries

## 🛠️ Tech Stack
- MySQL

## 📂 Project Structure
- schema.sql → Database & table creation
- data.sql → Basic data insertion
- data_generator.sql → Generate 1000 records
- queries.sql → CRUD operations
- joins.sql → Advanced SQL joins

## 🔗 Key Features
- ✔ CRUD Operations
- ✔ Data generation using SQL
- ✔ INNER JOIN, LEFT JOIN, RIGHT JOIN
- ✔ Aggregation with GROUP BY

## ▶️ How to Run
1. Run schema.sql  
2. Run data_generator.sql (or data.sql)  
3. Run queries.sql  
4. Run joins.sql  

## 📊 Sample Queries
- Students with marks > 60  
- Students enrolled in courses  
- Count students per course  

## 🚀 Future Improvements
- Window functions (RANK, DENSE_RANK)
- Stored procedures
- Indexing for performance
